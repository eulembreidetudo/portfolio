# Rakan Portfolio v2

Site portfolio em HTML, CSS e JavaScript puro, com estética calma, clean, bege/cinza/roxo acinzentado e inspiração visual urbana.

## Como usar

1. Abra `index.html` no navegador.
2. Troque `img/profile-placeholder.svg` por uma foto sua, se quiser. O ideal é manter o nome `profile-placeholder.svg` ou alterar o caminho no HTML.
3. Troque as imagens dos projetos:
   - `img/rako-ai.svg`
   - `img/lost-luggages.svg`
4. Coloque a música em:
   - `sounds/and-i-love-her.mp3`

O áudio começa apenas depois do clique em `entrar`, porque os navegadores bloqueiam autoplay com som sem interação do usuário.

## Arquivos principais

- `index.html` — estrutura do site
- `css/styles.css` — visual, responsividade e animações
- `js/main.js` — interações, áudio, scroll, hover, clipboard e reveals
- `img/gorillaz-bg.png` — imagem de fundo enviada por você
